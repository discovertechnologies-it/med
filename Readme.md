🧩 End-User Modules (Medicine eCommerce App)

1. 👤 Account & Authentication
   Sign up / Login (OTP / email)
   Profile details (name, age, gender)
   Manage saved addresses
   Manage prescriptions (upload/view/delete)
2. 🔍 Medicine Search & Discovery
   Search by:
   Medicine name (e.g., Dolo 650)
   Salt/composition (important feature)
   Filters:
   Price range
   Brand vs generic
   Prescription required / not
   Categories:
   Tablets, Syrups, Devices, Wellness

👉 UX tip: autocomplete + typo tolerance is very useful here

3. 💊 Medicine Details Page
   Medicine info:
   Name, composition, usage
   Manufacturer
   Dosage info
   Price & discounts
   Availability (in stock / out of stock)
   “Requires prescription” indicator
   Alternatives (generic substitutes)
4. 📷 Prescription Upload & Management
   Upload prescription (camera / file)
   Attach prescription to order
   View previously uploaded prescriptions
   Status:
   Pending / Approved / Rejected
5. 🛒 Cart Management
   Add/remove medicines
   Quantity control
   Validation:
   Block checkout if prescription missing
   Price breakdown:
   Total, discounts, delivery fee
6. 💳 Checkout & Payments
   Select address
   Choose delivery option:
   Home delivery / pickup
   Apply coupons
   Payment methods:
   UPI / Card / COD
7. 📦 Order Management
   Place order
   Order timeline:
   Confirmed → Packed → Shipped → Delivered
   Cancel / return (if allowed)
   Download invoice
8. 🚚 Real-Time Tracking
   Track delivery status
   Estimated delivery time
   Delivery partner info
   Live tracking (optional advanced)
9. 🔔 Notifications & Alerts
   Order updates
   Prescription approval/rejection
   Payment confirmation
   Refill reminders (very important for retention)
10. 🔁 Reorder & Subscription
    “Order again” from history
    Monthly medicine subscription
    Auto reminders before refill
11. ⭐ Reviews & Ratings
    Rate medicines/products
    Write reviews
    View other user feedback
12. ❤️ Wishlist / Save for Later
    Save medicines for future
    Quick add to cart later
13. 🧠 Smart Suggestions (Optional but powerful)
    Show:
    Cheaper generic alternatives
    Frequently bought together
    Based on:
    User history
    Popular products
14. 📍 Location & Availability
    Detect user location
    Show deliverable medicines
    Delivery ETA based on location
15. 🆘 Help & Support
    FAQ section
    Chat support / WhatsApp support
    Order issue reporting
    🎯 Clean User Flow (Important)

User journey should be:

Search → View Medicine → Add to Cart → Upload Prescription (if needed) → Checkout → Track Order → Reorder
